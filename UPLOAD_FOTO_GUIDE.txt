=========================

1. TAMBAH PRODUK BARU:
   - Klik menu "Produk" ‚Üí "Tambah Produk"
   - Isi semua field yang diperlukan
   - Pada field "Foto Produk", klik "Pilih File"
   - Pilih gambar dari komputer Anda
   - Format yang diterima: JPG, JPEG, PNG, GIF
   - Ukuran maksimal: 2MB
   - Klik "Simpan Produk"

2. EDIT PRODUK (GANTI FOTO):
   - Klik menu "Produk"
   - Klik icon "Edit" (pensil kuning)
   - Foto saat ini akan ditampilkan di bawah form
   - Jika ingin ganti foto, klik "Pilih File" pada field "Foto Produk"
   - Pilih foto baru
   - Foto lama akan otomatis dihapus saat menyimpan
   - Klik "Simpan Perubahan"

3. LOKASI PENYIMPANAN FOTO:
   - Semua foto disimpan di folder: assets/img/uploads/
   - Nama file akan otomatis dibuat (tidak perlu diubah)
   - Pastikan folder uploads memiliki izin baca-tulis

4. LIHAT FOTO PRODUK:
   - Di halaman "Data Produk", foto akan ditampilkan di kolom "Foto"
   - Ukuran display: 60x60 pixel
   - Jika tidak ada foto: akan ditampilkan icon gambar

5. SPESIFIKASI TEKNIS:
   - Format: JPG, JPEG, PNG, GIF
   - Ukuran maksimal: 2MB (2000KB)
   - Resolusi: Tidak ada batasan
   - Disarankan: Minimal 400x400 pixel untuk hasil terbaik
   - Tipe file: Raster image
